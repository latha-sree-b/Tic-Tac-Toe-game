# Tic-Tac-Toe AI 🤖

An interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**, featuring an **unbeatable AI opponent powered by the Minimax algorithm**. The project includes a modern glassmorphism user interface, smooth animations, dark/light theme toggle, and a live scoreboard for tracking wins, losses, and draws.

## Features

* 🎮 Play against an **unbeatable AI**
* 🧠 AI powered by **Minimax Algorithm**
* 🎨 Modern responsive UI with **glassmorphism styling**
* 🏆 Live scoreboard (Player vs AI vs Draws)
* ✨ Winning cell animations
* 🔄 Restart game anytime

## Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & animations
* **JavaScript (ES6)** – Game logic & AI implementation
